#  NLP Assignment – 3: Chatbot using Transformers

##  Overview
This project implements a **console-based conversational chatbot** using **pre-trained transformer models** from Hugging Face. The chatbot interacts with users in natural language and generates meaningful responses dynamically.

---

##  Objective
To build a simple AI chatbot that:
- Uses transformer-based models for text generation
- Maintains a continuous conversation with users
- Simulates a basic conversational AI assistant

---

##  Learning Outcomes
After completing this project, you will:
- Understand how transformer-based language models work  
- Use pre-trained models from Hugging Face Model Hub  
- Perform text generation using transformer models  
- Build an interactive conversational system  
- Understand prompt-based text generation  

---

##  Tech Stack
- **Programming Language:** Python  
- **Libraries:** Hugging Face Transformers  
- **Framework:** PyTorch / TensorFlow  
- **Environment:** Jupyter Notebook / VS Code
