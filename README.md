# 🧠 NLP Text Preprocessing Assignment

This project focuses on transforming messy, real-world textual data into clean, structured, and meaningful tokens suitable for machine learning models.

---

## 📌 Problem Statement

Raw text data often contains noise such as:
- Numbers
- URLs
- Repeated characters
- Extra spaces
- Irrelevant words

The goal is to build a preprocessing pipeline that cleans and standardizes such data for NLP tasks.
